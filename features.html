Current Features

Profile Management (Personalised Practices)

Student user can specify areas of study to focus on

System will suggest test papers based on profile

Interactive Tests

Digital assessment sheets from assessment titles by Marshall Cavendish Education

Dynamic papers consisting of questions selected by system based on user profile

Auto-marking feature - results tabulated on completion of tests

Model answers and worked solutions provided*

* Reference answers to be provided for Open-Ended questions
Performance Evaluation

Detailed history of past test attempts and results

System will highlight weaknesses and suggest tests to counter them

Other Features

User can tag a test or question as his favorite

System will highlight, on weekly basis, top 3 favorites and top 3 questions
      (with most incorrect attempts) across all users
